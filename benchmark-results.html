Key Takeaways

  • Gemini 3 Flash Preview significantly outperforms 2.5 Flash
  • Cairo Coder delivers massive gains on code generation:
    • 2.5 Flash: +19% absolute (46% → 65%)
    • 3 Flash: +23% absolute (67% → 90%)
  • Starknet Agent sees even larger improvements:
    • 2.5 Flash: +55.3% absolute (36% → 91.3%)
    • 3 Flash: +12.3% absolute (83.2% → 95.5%)
  • Optimization adds +5-15% on top of base pipeline
